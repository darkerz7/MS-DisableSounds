# MS-DisableSounds for ModSharp
Allows you to turn off loud sounds

## Required packages:
1. [ModSharp](https://github.com/Kxnrl/modsharp-public)
2. [ClientPreferences](https://github.com/Kxnrl/modsharp-public/tree/master/Sharp.Modules/ClientPreferences)
3. [LocalizerManager](https://github.com/Kxnrl/modsharp-public/tree/master/Sharp.Modules/LocalizerManager)

## Installation:
1. Install `ClientPreferences` and `LocalizerManager`
2. Compile or copy MS-DisableSounds to `sharp/modules/MS-DisableSounds` folger
3. Copy DisableSounds.json to `sharp/locales` folger

## Commands:
Client Command | Description
--- | ---
`ms_loud` | Toggles loud weapon sounds (KnifeHit, Scope, R8, AutoSwitch)
`ms_hits` | Toggles hit sounds (Damage)
`ms_footsteps` | Toggle footstep sounds
